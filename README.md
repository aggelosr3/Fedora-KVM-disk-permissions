# Fedora-KVM-disk-permissions
This script is a solution to a problem I encountered. I was unable to store virtual machines (VMs) on a drive other than my main system drive where Fedora is installed. It appears to be working.

I created everything required for KVM, but I kept getting a “permission denied” error.

This issue applies to all drives if they are not properly mounted.

The script is still in testing phase, as I was trying to fix the KVM-related problem.

If you decide to try it, you do so at your own risk. I strongly recommend testing it inside a VM first, if possible.
